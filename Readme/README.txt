# 🧠 Deep Learning Image Classification Project

## 📌 Overview

This project focuses on building and comparing multiple deep learning models for image classification. It includes model training, evaluation using confusion matrices, and organized storage of code, models, and results.

---

## 📁 Project Structure

```bash
majorfinalproject/
│
├── code/                  # All model training notebooks (ResNet, DenseNet, MobileNet)
├── models/                # Saved trained model files (.pth)
├── confusionmatrix/       # Confusion matrix output images
├── Readme/                # Project documentation files
├── label_map.json         # Class label mapping
```

---

## 🚀 Features

* Multiple CNN architectures implemented:

  * ResNet18 / ResNet50
  * DenseNet121
  * MobileNetV2
* Organized project structure
* Confusion matrix visualization
* Model saving and reuse
* Label mapping using JSON
